# Standup

一个很小的 macOS 菜单栏应用：

- 连续清醒 30 分钟后弹出站立提醒
- 提醒窗口自带 30 秒倒计时
- 屏幕休眠、电脑休眠、锁屏或解锁后会重新开始计时
- 构建后产物是独立 `.app`，不需要靠终端常驻

## 开发与构建

```bash
swift build
./Scripts/build-app.sh
open build/Standup.app
```

## 当前实现

- 菜单栏常驻图标，点击可看到剩余时间
- 到点弹出浮动提醒窗
- 可手动“立即重新计时”
- 可手动“立即显示提醒”做测试

## 登录时启动

当前版本没有内置登录项注册。若你希望开机自启，可以先手动把 `build/Standup.app` 加到“系统设置 -> 通用 -> 登录项”。
